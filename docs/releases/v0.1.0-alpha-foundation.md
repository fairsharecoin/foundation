# v0.1.0-alpha-foundation

Initial public alpha foundation release for FairShareCoin.

## Highlights
- Local-first wallet/account foundation with register/login/transfer/history/settings flows
- Security baseline: CSRF protections, session controls, rate limiting, cooldowns
- Verification lifecycle scaffolding and telemetry foundation
- Alpha operational preflight gates for staging/production-like configurations
- Governance/open-source baseline docs (security policy, contributing, code of conduct, license)

## Included in this release
- Foundation app source (`src/`)
- Tests (`test/`)
- CI workflow (`.github/workflows/ci-security.yml`)
- Operational and security docs (`docs/`, `OPEN_SOURCE_READINESS.md`)

## Alpha caveats
- This is not the final decentralized protocol runtime.
- Production infrastructure hardening and external review are ongoing.
- Identity verification provider integration remains staged.

## Upgrade/usage notes
- Use env templates (`.env.example`, `.env.staging.example`, `.env.production.example`)
- Run:
  - `npm run build`
  - `npm test`
  - `npm run preflight:staging`
  - `npm run preflight:prod`

## Security
Please report vulnerabilities via `SECURITY.md`.
